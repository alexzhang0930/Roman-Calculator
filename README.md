# Roman-Calculator

Calculator that uses Roman Numerals and Latin Operators

Syntax:
   Each input must be separated by spaces.
   Each operation must be surrounded by parentheses
   Ex: ( XII additur ( V multiplicatur II ) )

Key:
   "+" = "additur"
   "-" = "demitur"
   "*" = "multiplicatur"
   "/" = "dividitur"

Extra Notes:
   Use "q" to quit program
